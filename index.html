Comment synchroniser vos données entre plusieurs postes

  1. Sur le poste source, exportez les données en cliquant sur "Exporter les données"
  2. Enregistrez le fichier JSON généré à un emplacement accessible
  3. Transférez ce fichier vers l'autre poste (par email, clé USB, réseau, etc.)
  4. Sur le poste de destination, cliquez sur "Parcourir" et sélectionnez le fichier JSON
  5. Cliquez sur "Importer les données" pour charger les données sur le nouveau poste
  6. Répétez ce processus chaque fois que vous modifiez les données sur un poste

Note : Pour une synchronisation automatique, vous devriez envisager d'utiliser une base de données centralisée ou un service cloud.
